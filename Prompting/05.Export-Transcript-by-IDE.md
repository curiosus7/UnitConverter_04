## TD-10 PyQt — 제안 UI (말씀하신 구성)

```
┌─────────────────────────────────────────┐
│  Unit:     [ meter ▼ ]                  │
│  Value:    [ 2.5        ]               │
│  Format:   [ table ▼ ]   (json/csv/table)│
│            [ Convert ]                  │
├─────────────────────────────────────────┤
│  meter: 2.5                             │
│  feet: 8.2021                           │
│  yard: 2.7340                           │
│  ...                                    │
└─────────────────────────────────────────┘
         ↑ QPlainTextEdit / QTextEdit
```

| 위젯 | 역할 | 백엔드 호출 |
|------|------|-------------|
| `QComboBox` (Unit) | 입력 단위 선택 | Registry 단위 목록 |
| `QLineEdit` / `QDoubleSpinBox` (Value) | 숫자 입력 | `input_parser` 검증 (음수·형식) |
| `QComboBox` (Format) | json / csv / table | `output_formatter.write_*` (TD-09) |
| `QPushButton` (Convert) | 변환 트리거 | `converter.convert_all()` |
| `QPlainTextEdit` (결과) | 읽기 전용 출력 | formatter → `StringIO` → setPlainText |

오류(E-FMT, E-NUM, E-NEG, E-UNIT)는 **결과 창에 메시지 표시** 또는 **QMessageBox** — lecture Boundary TC로 고정하면 됩니다.

5. **`input_parser` L12** — 콜론 없는 입력 E-FMT (Domain D-PAR-01은 subprocess)
