## Coverage 보는 방법

프로젝트 루트에서 가상환경을 활성화한 뒤:

### 1. 터미널 요약 (가장 빠름)

```bash
pytest tests/ --cov=unit_converter --cov-report=term-missing
```

- `--cov=unit_converter` — `unit_converter/` 패키지만 측정
- `--cov-report=term-missing` — 파일별 커버리지 + **미커버 라인 번호** 출력

### 2. HTML 리포트 (시각적으로 보기 좋음)

```bash
pytest tests/ --cov=unit_converter --cov-report=html
```

생성 후 브라우저에서 `htmlcov/index.html`을 엽니다.  
(`.gitignore`에 `htmlcov/`, `.coverage`는 이미 제외되어 있습니다.)

### 3. 브랜치 커버리지까지 (선택)

```bash
pytest tests/ --cov=unit_converter --cov-branch --cov-report=term-missing
```
